Manual de utilização dos algoritmos de
Busca em Java

Introdução

Este manual explica como utilizar o pacote de algoritmos de busca implementados em Java. Com este pacote é possível resolver um problema de busca especificando-se apenas a classe que representa os estados, os algoritmos de busca

  • largura,
  • profundidade,
  • profundidade iterativa,
  • bi-direcional,
  • heurística e
  • subida da montanha
já estão implementados.

A interface Estado

A classe que representa o "estado" para cada problema específico deve implementar a interface Estado que tem os seguintes métodos:

  • O método getDescricao() deve retornar uma string descrendo o problema. Esse método é eventualmente utilizado pela interface do programa.

  • O método ehMeta() deve retornar true se o estado for o estado meta.

  • O método custo() retorna o custo de gerar o estado (não o custo acumulado, este é calculado no Nodo da árvore).

  • O método sucessores() deve retornar uma lista de estados sucessores.

Exemplo

Para demonstrar a utilização do pacote, será resolvido o problema do O homem, o lobo, o carneiro e o cesto de alface, que tem o seguinte enunciado:

Uma pessoa, um lobo, um carneiro e um cesto de alface estão à beira de um rio. Dispondo de um barco no qual pode carregar apenas um dos outros três, a pessoa deve transportar tudo para a outra margem. Determine uma série de travessias que respeite a seguinte condição: em nenhum momento devem ser deixados juntos e sozinhos o lobo e o carneiro ou o carneiro e o cesto de alface.

Para resolver este problema, os estados devem ter as seguintes informações:

  • Em que lado do rio estão cada um dos personagens (o homem, o lobo, o carneiro e o cesto de alface).
  • Estrutura de dados: quatro variáveis caracter onde 'd' representa lado direito e 'e' lado esquerdo. 

    char homem, lobo, carneiro, alface;

  • Estado inicial: 
    homem='e', lobo='e', carneiro='e', alface='e';

  • Estado meta: 
    homem='d', lobo='d', carneiro='d', alface='d';

A geração de sucessores pode ser resumida como

  • "levar x da margem w para a margem zbeginitemize
  • x é uma das variáveis { carneiro, alface, lobo } ou não levar nada,
  • w é 'd' ou 'e',
  • z é 'd' ou 'e'.
Pré-condições
  • w é diferente de z,
  • o homem estar na margem w,
  • o homem estar na mesma margem de x,
  • não ficar na mesma margem ((o lobo e o carneiro) ou (o carneiro e o alface)) sem o homem.

Efeitos

  • Se x é 'd', vira 'e'.
  • Se x é 'e', vira 'd'.
  • Se homem é 'd', vira 'e'.
  • Se homem é 'e', vira 'd'.

001 package exemplos;
002 
003 import java.util.LinkedList;
004 import java.util.List;
005 
006 import busca.BuscaLargura;
007 import busca.Estado;
008 import busca.Nodo;
009 
010 /**
011  * Representa um estado do mundo: em que lado estao
012  * o homem, o lobo, o alface e o carneiro
013  */
014 public class HLAC implements Estado {
015     
016     public String getDescricao() {
017         return
018         "Uma pessoa, um lobo, um carneiro e um cesto de alface estao a beira \n"+
019         "de um rio. Dispondo de um barco no qual pode carregar apenas um dos \n"+
020         "outros tres, a pessoa deve transportar tudo para a outra margem. \n"+
021         "Determine uma serie de travessias que respeite a seguinte condicao: \n"+
022         "em nenhum momento devem ser deixados juntos e sozinhos o lobo e o \n"+
023         "carneiro ou o carneiro e o cesto de alface. \n\n";
024     }
025     
026     final char homem, lobo, carneiro, alface; // preferir "immutable objects"
027     final String op; // operacao que gerou o estado
028     
029     /** construtor para o estado, recebe cada valor de atributo */
030     public HLAC(char h, char l, char c, char a, String o) {
031         homem = h;
032         lobo = l;
033         carneiro = c;
034         alface = a;
035         op = o;
036     }
037     
038     /**
039      * verifica se o estado e meta
040      */
041     public boolean ehMeta() {
042         return homem=='d' && lobo=='d' && carneiro=='d' && alface=='d';
043     }
044     
045     /**
046      * Custo para geracao do estado
047      */
048     public int custo() {
049         return 1;
050     }
051     
052     /**
053      * gera uma lista de sucessores do nodo.
054      */
055     public List<Estado> sucessores() {
056         List<Estado> suc = new LinkedList<Estado>(); // a lista de sucessores
057         
058         levarCarneiro(suc); // tenta levar o carneiro
059         levarLobo(suc);
060         levarAlface(suc);
061         levarNada(suc);
062         
063         return suc;
064     }
065     
066     private void levarCarneiro(List<Estado> suc) {
067         if (carneiro == homem) { // homem e carneiro na mesma margem
068           char novaMargem = ladoOposto(homem);
069             HLAC novo = new HLAC(novaMargem, lobo, novaMargem,  alface, "levarC-"+carneiro+novaMargem);
070             if (novo.ehValido()) {
071                 suc.add(novo);
072             }
073         }
074     }
075     
076     private void levarLobo(List<Estado> suc) {
077         if (lobo == homem) {
078           char novaMargem = ladoOposto(homem);
079             HLAC novo = new HLAC(novaMargem, novaMargem, carneiro,  alface, "levarL-"+lobo+novaMargem);
080             if (novo.ehValido()) {
081                 suc.add(novo);
082             }
083         }
084     }
085     
086     private void levarAlface(List<Estado> suc) {
087         if (alface == homem) {
088           char novaMargem = ladoOposto(homem);
089             HLAC novo = new HLAC(novaMargem, lobo, carneiro, novaMargem, "levarA-"+alface+novaMargem);
090             if (novo.ehValido()) {
091                 suc.add(novo);
092             }
093         }
094     }
095     
096     private void levarNada(List<Estado> suc) {
097       char novaMargem = ladoOposto(homem);
098         HLAC novo = new HLAC(novaMargem, lobo, carneiro, alface, "levarN-"+homem+novaMargem);
099         if (novo.ehValido()) {
100             suc.add(novo);
101         }
102     }
103 
104     private char ladoOposto(char l) {
105       if (l == 'e') {
106         return 'd';
107       } else {
108         return 'e';
109       }
110     }
111     
112     
113     /** returna true se o estado e valido */
114     private boolean ehValido() {
115         if (lobo == carneiro && lobo != homem) {
116             // lobo e carneiro estao na mesma margem sem o homem
117             return false;
118         }
119         if (carneiro == alface && carneiro != homem) {
120             // lobo e carneiro estao na mesma margem sem o homem
121             return false;
122         }
123         return true;
124     }
125     
126     /**
127      * verifica se um estado e igual a outro
128      * (usado para poda)
129      */
130     public boolean equals(Object o) {
131         if (o instanceof HLAC) {
132             HLAC e = (HLAC)o;
133             return homem==e.homem && lobo==e.lobo && carneiro==e.carneiro && alface==e.alface;
134         }
135         return false;
136     }
137     
138     /** 
139      * retorna o hashCode desse estado
140      * (usado para poda, conjunto de fechados)
141      */
142     public int hashCode() {
143         return (""+homem + lobo + carneiro + alface).hashCode();
144     }
145     
146     public String toString() {
147         String dir = ""; // quem esta na dir
148         String esq = ""; // quem esta na esq
149         if (homem == 'd') {
150             dir += 'h';
151         } else {
152             esq += 'h';
153         }
154         if (lobo == 'd') {
155             dir += 'l';
156         } else {
157             esq += 'l';
158         }
159         if (carneiro == 'd') {
160             dir += 'c';
161         } else {
162             esq += 'c';
163         }
164         if (alface == 'd') {
165             dir += 'a';
166         } else {
167             esq += 'a';
168         }
169         return esq +"|" + dir + " (" + op + ")\n";
170     }

173     public static void main(String[] a) {
174         HLAC inicial = new HLAC('e', 'e', 'e', 'e', "inicial");
175         
176         // chama busca em largura
177         System.out.println("busca em largura");

179         if (n == null) {
180             System.out.println("sem solucao!");
181         } else {
182             System.out.println("solucao:\n" + n.montaCaminho() + "\n\n");
183         }
184     }

Compilação e execução

As classes do pacote estão no arquivo buscaJava.jar, portanto este arquivo deve estar no classpath.

Para windows: 

  set CLASSPATH=C:\xxx\buscaJava.jar;.
  javac HLAC.java
  java HLAC

Para unix: 

  export CLASSPATH=/xxx/buscaJava.jar:.
  javac HLAC.java
  java HLAC

Utilização dos algoritmos de busca

Largura, Profundidade e Profundidade Iterativo

As classes que implementam a interface Estado podem utilizar os seguintes algoritmos de busca

  • new BuscaLargura().busca( Estado inicial ): recebe o estado inicial e retorna o Nodo da árvore de busca que é a solução do problema. Com o método montaCaminho() da classe Nodo, pode-se imprimir a solução. Retorna null se não encontrar a solução.

    No método main está exemplificada a utilização destes algoritmos.

  • new BuscaProfundidade(int max).busca( Estado inicial ): recebe o estado inicial e a profundidade máxima e retorna o Nodo solução.

  • new BuscaRecursiva(int max).busca( Estado inicial ): implementação recursiva (sem lista de abertos) para a busca em profundidade.

  • new BuscaIterativo().busca( Estado inicial ): implementação do busca em profundidade iterativo.

Mais informações sobre a classe Nodo e como ela pode ser utilizada podem ser obtidas na API do pacote e/ou nos programas exemplo (diretório src/exemplos).

Busca Bi-Direcional

Para a utilização do algoritmo de busca bi-direcional, invocado com new BuscaBidirecional().busca( Estado inicial, Estado meta ), a classe que representa o estado também deve implementar a interface Antecessor. Esta interface tem apenas um método:

  • antecessores() que retorna uma lista de estados antecessores de um determinado estado.

No exemplo 8-Puzzle esta interface é implementada.

Busca A*

Para a utilização do algoritmo de busca A*, invocado com new AEstrela().busca( Estado inicial ), a classe que representa o estado também deve implementar a interface Heuristica. Esta interface tem o seguinte método:

  • h() retorna a estimativa de custo para transformar um estado no estado meta;
Nos seguintes exemplos essa interface é implementada
  • 8-Puzzle.
  • N-Rainhas.
  • Quadrado mágico.

Busca Subida da Montanha

Para a utilização do algoritmo de busca Subida da Montanha, invocado com new SubidaMontanha().busca( Estado inicial ), a classe que representa o estado também deve implementar a interface Aleatorio. Esta interface tem o seguinte método:

  • geraAleatorio() que gera um estado aleatóriamente.
Nos seguintes exemplos essa interface é implementada
  • N-Rainhas.
  • Quadrado mágico.
